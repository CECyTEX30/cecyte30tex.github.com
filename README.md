# CECyTE 30 Texmolac - Página Informativa

¡Bienvenido al repositorio oficial del proyecto web del **CECyTE 30**! 

Esta página ha sido desarrollada como un espacio informativo para la comunidad escolar, permitiendo consultar datos importantes, especialidades, instalaciones y materiales audiovisuales de nuestra institución.

## 🚀 Características principales
- **Presentación:** Información general del plantel.
- **Video:** Acceso directo a nuestro material audiovisual de presentación.
- **Especialidades:** Detalle sobre las áreas académicas ofrecidas.
- **Instalaciones:** Galería sobre nuestras instalaciones.

## 🛠️ Tecnologías utilizadas
- **HTML5:** Estructura del sitio web.
- **CSS3:** Diseño y maquetación (estilos visuales).

## 📂 Estructura del proyecto
- `/index.html`: Página de inicio.
- `/paginas/`: Contiene el resto de las secciones del sitio.
- `/imagenes/`: Recursos visuales del sitio.
- `/videos/`: Material audiovisual.
